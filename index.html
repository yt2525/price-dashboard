ℹ How does price grabbing work?

Four files in your GitHub repo work together:

  1. urls.json — the list of competitor URLs you defined here. Click ⬇ urls.json to download, then commit to repo.
  2. scrape.py — Python script with one "adapter" per competitor. Each adapter knows how to read prices from that site's HTML.
  3. scrape.yml — GitHub Actions workflow that runs every 6 hours on a free server. Visits each URL, extracts prices, writes prices.json.
  4. prices.json — the captured prices. Your dashboard fetches it on load and populates the grid.

Real-time scraping from the browser isn't possible — CORS blocks it. That's why this runs server-side via GitHub Actions on a 6-hour cadence. To force an immediate update, click ▶ Trigger Scraper below.
